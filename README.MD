## 요구사항
- 반드시 하루에 한 아이당 1타임(2시간)만 가능
- 한 부모님당 3개의 예약 가능
- 예약 가능 일자는 오늘로부터 7일 뒤까지(평일은 제외)
- 11세까지만 입장
- 한 타임에 30명
